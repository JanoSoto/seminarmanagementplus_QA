Pre-cerrar Semestre

¿Está seguro? Esta acción generará los siguientes cambios dentro del sistema.
  • Los borradores finales pasan a pertenecer a este semestre que se pre-cerrará.
  • Los temas en proceso de examen pasan a pertenecer a este semestre que se pre-cerrará.
  • Aquellos borradores finales entregados después que se precerró el semestre actual quedarán para el semestre siguiente.
  • Aquellos temas vigentes con borrador final que cambien a proceso de examen entre el pre-cierre y cierre quedarán para el semestre siguiente.
  • Aquellos temas vigentes que cambien a prorrogados entre el pre-cierre y cierre quedarán para el semestre siguiente.
